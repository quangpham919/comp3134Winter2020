What is the definition of server port ? 
- A port number is a way to identify a specific process to which an Internet or other network
 message is to be forwarded when it arrives at a server. For For the Transmission Control Protocol and the User Datagram Protocol, a port number is a 16-bit integer that is put in the header appended to a message unit.

What is server port range ? 
- Port numbers range from 0 to 65535, but only port numbers 0 to 1023 are reserved for privileged services and designated as well-known ports

How many bits does number above represent ? 
- 16 bits
